# Transformer Lab

Transformer Lab is an open-source research initiative. Our goal is to make the tools for advanced AI research open and accessible, so that anyone can run any machine learning workload — from training and fine-tuning to evaluation — with the same quality of tools currently reserved for the largest frontier labs.
